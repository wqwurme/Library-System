# Library-System



ЧТОБЫ НАЧАТЬ РАБОТУ НУЖНО ЧЕТКО СЛЕДОВАТЬ ИНСТРУКЦИИ:

1.Устанавливаем python 
2.В файлах models/NewBook.py ; models/User.py ; models/loan.py обязательно следует изменить пути к файлам на АБСОЛЮТНЫЕ ПУТИ К ВАШЕМУ ФАЙЛУ
3.Запускаем main.py через терминал командой python "твой абсолютный путь к файлу"
4.Готово! Теперь ты можешь добавлять книги пользователей и вести полный учет.
